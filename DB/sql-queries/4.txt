FÃ¼r welche Produkte ist das teuerste Angebot mehr als doppelt so teuer wie das preiswerteste?



Query
-----------
SELECT produkt_id
FROM produkt
WHERE 
  (SELECT MAX(preis) 
    FROM angebot 
    NATURAL JOIN angebotsdetails 
    WHERE angebot.produkt_id = produkt.produkt_id
  ) > 2 * (SELECT MIN(preis)
            FROM angebot
            NATURAL JOIN angebotsdetails 
            WHERE angebot.produkt_id = produkt.produkt_id
          );
